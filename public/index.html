API REST 8B1


Explicación:

ApiRest se creó para dar respuestas a las peticiones de la tienda web de la
materia de sistemas distribuidos, son 8 rutas a donde enviar las peticiones.

Cada ruta se encarga de devolver información valiosa para el funcionamiento
del sistema.

El formato en que se deben enviar datos en las peticones es JSON así también
en este formato recibirás las respuestas.

Insertar productos

Recibe un JSON con los datos del nuevo producto y lo inserta en la base de datos

https://st0re.herokuapp.com/api/productos/insertar

 Comprar producto

Recibe un JSON con los datos del producto, resta uno a la base de datos, lo agrega al carrito, lo agrega al historial de usuario, solicita un retiro al banco

https://st0re.herokuapp.com/api/productos/comprar

 Listado de productos

Recibe una petición get y retorna todos los productos de la tienda

https://st0re.herokuapp.com/api/productos/listado

 Añadir al carrito

Recibe un JSON con los datos de la semicompra y lo guarda en la BD

https://st0re.herokuapp.com/api/productos/semicompra

 Iniciar sesion

Recibe un JSON con los datos del supuesto usuario y verifica que exista

https://st0re.herokuapp.com/api/usuario/login

 Historial de compras del usuario

Recibe un JSON con los datos del usuario y busca todas sus compras

https://st0re.herokuapp.com/api/usuario/historial

 Carrito de compras del usuario

Recibe un JSON con los datos del usuario y busca todas sus productos en el carrito de compras

https://st0re.herokuapp.com/api/usuario/carrito

 Datos del usuario

Recibe un JSON con el nombre del usuario y busca todos los datos del usuario

https://st0re.herokuapp.com/api/usuario/datos





ApiRest se programó en nodeJS con el frameword express y está alojado
en un servidor gratuito de heroku.

La bases de datos usada es no relacional con MongoDB, alojada en un
cluster de Mongo Atlas DB.
